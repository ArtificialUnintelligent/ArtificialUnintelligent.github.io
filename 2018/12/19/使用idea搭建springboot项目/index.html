springboot是Pivotal团队提供的全新框架，其设计目的是为了简化Spring应用的初始搭建以及开发过程。框架使用了特定方式来进行配置，从而使开发人员不再需要定义样板化的配置。

创建流程：

  1. 新建项目

  1. 选择spring initializr，点击next

  2. 填写详细项目信息，packaging-打包方式默认为jar包，如果需要在tomcat下进行部署，可选择war包，后面会有详细配置说明

    ​ Name：项目名称

    　　　　　　Type：我们是Maven构建的，那么选择第一个Maven Project

    　　　　　　Packaging：打包类型，打包成Jar文件

    　　　　　　Java Version：jdk版本，选择1.8

    　　　　　　Language：开发语言，选择Java

    　　　　　　Group：对应pom文件中的groupId，项目组织的唯一标识，对应Java包的结构

    　　　　　　Artifact：对应pom文件的artifactId，项目唯一标识，对应项目名称

    　　　　　　Version：项目版本，对应pom文件的version

    　　　　　　Description：项目描述，对应pom文件的description

    　　　　　　Package：包名

  3. 设置springboot版本和项目类型

  1. 项目名称和路径，填完点击finish完成新建项目

  2. 带maven引入所有依赖即可看到项目基本的雏形


以下是一些注意事项：

  1. 通常springboot使用jar包进行项目的部署运行，使用mvn clean install进行打包，运行jar包时，在jar包目录下控制台输入命令java -jar xxx.jar即可，如果需要指定端口，在application.properties里输入server.port=来指定端口

  2. springboot部署运行jar包后访问路径为localhost:xxxx(指定的端口号)/接口路径，不需要在路径中加入包名

  3. 引入本地jar包

    先在src同级目录下新建lib目录，将本地jar包放在lib下，在pom文件中引入jar包依赖
