先说redis是神马，用一大串的形容词来描述redis就是：开源的、使用ANSIC语言编写、支持网络、可基于内存亦可持久化的日志型、Key-Value数据库，没有错，这是我从百度百科上复制粘贴过来的，所以说redis是什么呢？是一个k-v的关系型数据库

redis提供了多种语言的api，在java项目开发中也有广泛使用


Redis支持的数据结构

  • String
  • Hash
  • List
  • Set
  • Sorted Set


Redis的简单使用

  • 远程连接redis：在安装有redis的机器上使用redis-cli -h [host] -p [port] 然后输入密码即可连接对应的数据库

  • 使用info指令查看当前redis相关信息

  • 使用select 0（或其他对应数字）连接db

  • 执行keys 可查看所有数据，支持三种通配符：? []

    例子：查看所有中间包含aaa的key

  • 操作：add key value 新增数据

    ​ del key 删除数据

Redis for mac

  • 下载redis安装包：https://redis.io/download，下载后解压
  • cd到redis-3.2.9目录，执行3个命令，make->make install->make test；期间会让输入电脑密码
  • redis-cli验证是否能连接到redis
